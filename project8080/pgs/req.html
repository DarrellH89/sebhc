H8 Emulator Hosting Requirements

    The emulator requires Macintosh OS Release 8.1 or better. Version 5 was developed using Apple's MPW version 3.6a1.

    A Power Macintosh is required.

    A color monitor is required.

    The Appearance Manager is required; the emulator will not start if it is not present.

    The emulator is a relatively small application that requires only about 2 megabytes of disk space, as initially distributed, and 5 megabytes of memory at run time.

Performance

    The author runs the emulator on a 300mhz G3 desktop system. Performance is supurb.

    User testing suggests that a 7100/66 is about the minimum required to get a good emulation experience at an emulated 2mhz clock speed. Any Mac faster than that should work fine.

    As host system speed increases, you will see more benefit from the faster clock speeds available on the emulator (ref: Emulator Menus). Please refer to the Design section of the project web site for details.
